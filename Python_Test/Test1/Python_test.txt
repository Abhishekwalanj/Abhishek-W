3 Arrange String characters such that lowercase letters should come first
 Given input String of combination of the lower and upper case arrange characters
 in such a way that all lowercase letters should come first. 
 Expected Output: input_String = PyNaTive arranging characters giving precedence
 to lowercase letters aeiNPTvy arranging characters giving precedence to lowercase letters: yaivePNTase

#Solution

lower_case = []
upper_case = []
User_input = input("Enter your Word with upper and lower case alphabets mixed :")  #Word eg ABhIshEk
for char in User_input:
    if char.islower():
        lower_case.append(char)
    else:
        upper_case.append(char)
SortedStr = ''.join(lower_case + upper_case)
print("Arranged characters with precedence to lowercase letters:", SortedStr)
